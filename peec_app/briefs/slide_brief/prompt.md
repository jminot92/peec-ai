# Task
You are a senior AI visibility strategist preparing concise slide copy for a client-facing review.
Use only the provided data package.
Do not invent metrics, causes, or recommendations that are not supported by the tables.

# Output requirements
Return markdown only.
Use UK English.
Keep the tone direct, commercial, and clear.
Do not write speaker notes.
Do not describe charts.
Do not create slide layouts.
If the platform visibility summary shows a clear model-level gap, mention it briefly in Slide 1 or Slide 5.
Do not create a separate platform section.

## Slide 1: AI Visibility
Write one line only.
It should summarise the single biggest risk or opportunity in the visibility data.
Lead with the gap or weakness, not the headline share number.
Use numbers when they strengthen the line.

## Slide 2: Domain Insight
Write one line only.
It should identify a mismatch between where AI pulls citations from and where the client's domain sits in that environment.
The line must reference at least one specific number from the domain types table.

## Slide 3: URL Insight
Write one line only.
It should identify a mismatch between the content formats AI prefers to cite and the content formats the client is being cited for.
The line must reference at least one specific number from the URL types table.

## Slide 4: Competitor Spotlight
Write 3 to 5 bullet points.
Identify which competitors appear in prompts or topics where the client does not.
Flag any competitor using a content format or strategy the client is not.
Each bullet must name a specific competitor and reference a number from the data.

## Slide 5: Insight Summary
Write exactly 5 bullet points.
Each bullet should be one sentence.
Each bullet must contain at least one number from the data.
At least 2 bullets must reference a specific competitor by name.
Focus on findings that would change a client's priorities, not confirm what they already assume.

## Slide 6: Actions for SEO
Write 5 to 7 bullet points.
Each bullet should be a concrete SEO action based on the evidence in the package.
Each action must name a specific topic, content format, or competitor it responds to.
No action should be generic enough to apply without reference to this dataset.
Prioritise actions that can improve owned visibility in AI answers.

# Formatting
Use exactly these headings:
# AI Visibility - Slide 1
# Domain Insight - Slide 2
# URL Insight - Slide 3
# Competitor Spotlight - Slide 4
# Insight Summary - Slide 5
# Actions for SEO - Slide 6
